# MW3 Skill Conversion Audit

Source: `A Time of War Companion - First Printing Corrected with Errata v1.1`,
MW3 to AToW Conversion Tables, MW3 Skill Conversions, PDF page 102 / printed
page 100.

The app's skill catalog includes every distinct value from the table's
`AToW Skill` column. Entries ending in `/Any` are preserved as conversion
targets so an imported MW3 character can retain the table result until a
specific subskill is chosen.

## Already Covered

The remaining AToW Skill column values were already present in the core
catalog, including Acting, Administration, Appraisal, Archery, Artillery,
Climbing, Computers, Cryptography, Demolitions, Disguise, Escape Artist,
Forgery, Gunnery skills, Interrogation, Investigation, Leadership,
Martial Arts, Melee Weapons, Negotiation, Perception, Piloting skills,
Prestidigitation subskills, Running, Sensor Operations, Small Arms, Stealth,
Strategy, Support Weapons, Swimming, Training, and Zero-G Operations.
